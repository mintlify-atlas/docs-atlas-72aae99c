<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold dark:text-white text-gray-900 mb-4">Quick Start</h2>
  <p className="text-lg dark:text-gray-400 text-gray-600 mb-8">Get up and running with Integra in minutes</p>
  
  <Steps>
    <Step title="Authenticate with the API">
      Obtain a JWT token by sending your credentials to the authentication endpoint.
      
      ```bash cURL
      curl -X POST https://api.integra.example/auth/login \
        -H "Content-Type: application/json" \
        -d '{
          "username": "your-username",
          "password": "your-password"
        }'
      ```
      
      <Accordion title="Example Response">
        ```json
        {
          "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
          "type": "Bearer",
          "expiresIn": 3600
        }
        ```
      </Accordion>
    </Step>
    
    <Step title="Record Employee Attendance">
      Use the attendance endpoint to clock employees in and out.
      
      ```bash cURL
      curl -X POST https://api.integra.example/asistencia/iniciar \
        -H "Authorization: Bearer YOUR_TOKEN" \
        -H "Content-Type: application/json" \
        -d '{
          "empleadoId": 1234,
          "unidadId": 5,
          "foto": "base64_encoded_photo"
        }'
      ```
      
      Learn more in the [Attendance Recording Guide](/guides/attendance-recording).
    </Step>
    
    <Step title="Generate Reports">
      Export attendance data to Excel for payroll processing.
      
      ```bash cURL
      curl -X GET "https://api.integra.example/reportes/excel?fecha=2024-03-01&unidadId=5" \
        -H "Authorization: Bearer YOUR_TOKEN" \
        --output attendance-report.xlsx
      ```
      
      Explore the [Reports API](/api/reports/excel-export) for more options.
    </Step>
  </Steps>
</div>
