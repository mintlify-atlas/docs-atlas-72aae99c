## POST /auth/login

Authenticates a user with username and password credentials and returns a JWT token for subsequent API requests.

**Authentication**: None required (public endpoint)

**Source**: `LoginController.java:21`

### Request

<ParamField body="username" type="string" required>
  Username for authentication. Must be between 3 and 50 characters.
  
  **Validation**: 
  - Required field
  - Minimum length: 3 characters
  - Maximum length: 50 characters
</ParamField>

<ParamField body="password" type="string" required>
  User password. Must be between 1 and 100 characters.
  
  **Validation**:
  - Required field  
  - Minimum length: 1 character
  - Maximum length: 100 characters
</ParamField>

### Response

<ResponseField name="token" type="string">
  JWT authentication token. Include this token in the `Authorization` header as `Bearer {token}` for authenticated requests.
</ResponseField>

<ResponseField name="employeeName" type="object">

### Examples

<CodeGroup>
```bash cURL
curl -X POST https://api.integra.example.com/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "username": "jdoe",
    "password": "securePassword123"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.integra.example.com/auth/login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    username: 'jdoe',
    password: 'securePassword123'
  })
});

const data = await response.json();
console.log('Token:', data.token);
```

```python Python
import requests

response = requests.post(
    'https://api.integra.example.com/auth/login',
    json={
        'username': 'jdoe',
        'password': 'securePassword123'
    }
)

data = response.json()
print(f"Token: {data['token']}")
```

```java Java
HttpClient client = HttpClient.newHttpClient();
HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("https://api.integra.example.com/auth/login"))
    .header("Content-Type", "application/json")
    .POST(HttpRequest.BodyPublishers.ofString(
        "{\"username\":\"jdoe\",\"password\":\"securePassword123\"}"
    ))
    .build();

HttpResponse<String> response = client.send(request, 
    HttpResponse.BodyHandlers.ofString());
System.out.println(response.body());
```
</CodeGroup>

## Using the JWT Token

After successful authentication, include the JWT token in all subsequent API requests:

```bash
curl -X GET https://api.integra.example.com/api/employees \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

<Note>
  JWT tokens have an expiration time. When a token expires, the client must re-authenticate using this login endpoint to obtain a new token.
</Note>

<Warning>
  Always transmit credentials over HTTPS. Never log or expose JWT tokens in client-side code or version control.
</Warning>
