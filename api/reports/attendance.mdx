## Get Attendance Report

Retrieve detailed attendance records in JSON format.

### Endpoint

```http
GET /asistencia/reporte/asistencias
```

### Query Parameters

<ParamField query="empleadoId" type="integer" optional>
  Filter by specific employee ID
</ParamField>

<ParamField query="desde" type="datetime" optional>
  Start date/time in ISO 8601 format (e.g., 2026-03-01T00:00:00)
</ParamField>

<ParamField query="hasta" type="datetime" optional>
  End date/time in ISO 8601 format (e.g., 2026-03-31T23:59:59)
</ParamField>

<ParamField query="unidadId" type="integer" optional>
  Filter by organizational unit ID
</ParamField>

<ParamField query="empleadoResponsableId" type="integer" optional>
  Filter by responsible employee/manager ID
</ParamField>

<ParamField query="supervisorId" type="integer" optional>
  Filter by supervisor ID
</ParamField>

<ParamField query="zonaId" type="integer" optional>
  Filter by geographic zone ID
</ParamField>

<ParamField query="puestoId" type="integer" optional>
  Filter by job position ID
</ParamField>

### Response

<ResponseField name="data" type="array">
  <Expandable title="EmpleadoReporte">
    <ResponseField name="empleadoId" type="integer">
      Employee ID
    </ResponseField>
    
    <ResponseField name="nombreCompleto" type="string">
      Employee full name
    </ResponseField>
    
    <ResponseField name="nip" type="string">
      Employee identification number
    </ResponseField>
    
    <ResponseField name="puesto" type="string">
      Job position
    </ResponseField>
    
    <ResponseField name="departamento" type="string">
      Department name
    </ResponseField>
    
    <ResponseField name="asistencias" type="array">
      Array of attendance records for this employee
      <Expandable title="Asistencia">
        <ResponseField name="id" type="integer">
          Attendance record ID
        </ResponseField>
        
        <ResponseField name="fecha" type="string">
          Date of attendance (ISO 8601)
        </ResponseField>
        
        <ResponseField name="entrada" type="string">
          Clock-in time (HH:mm:ss)
        </ResponseField>
        
        <ResponseField name="salida" type="string">
          Clock-out time (HH:mm:ss)
        </ResponseField>
        
        <ResponseField name="horasTrabajadas" type="string">
          Total hours worked (HH:mm:ss)
        </ResponseField>
        
        <ResponseField name="pausas" type="array">
          Array of break periods
          <Expandable title="Pausa">
            <ResponseField name="inicio" type="string">
              Break start time (HH:mm:ss)
            </ResponseField>
            
            <ResponseField name="fin" type="string">
              Break end time (HH:mm:ss)
            </ResponseField>
            
            <ResponseField name="duracion" type="string">
              Break duration (HH:mm:ss)
            </ResponseField>
          </Expandable>
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string">
  Success message: "Asistencia por empleados"
</ResponseField>

### Example Request

```bash cURL
curl -X GET "http://localhost:8081/comialex/api/integra/asistencia/reporte/asistencias?empleadoId=456&desde=2026-03-01T00:00:00&hasta=2026-03-31T23:59:59" \
```

## Get Monthly Summary

Retrieve a summary of attendance for a specific employee and month.

### Endpoint

```http
GET /asistencia/reporte/resumen-mes
```

### Query Parameters

<ParamField query="empleadoId" type="integer" required>
  Employee ID
</ParamField>

<ParamField query="anio" type="integer" optional>
  Year (defaults to current year)
</ParamField>

<ParamField query="mes" type="integer" optional>
  Month (1-12, defaults to current month)
</ParamField>

### Response

<ResponseField name="data" type="object">
  <Expandable title="ResumenMesAsistencia">
    <ResponseField name="empleadoId" type="integer">
      Employee ID
    </ResponseField>
    
    <ResponseField name="anio" type="integer">
      Year
    </ResponseField>
    
    <ResponseField name="mes" type="integer">
      Month (1-12)
    </ResponseField>
    
    <ResponseField name="diasLaborados" type="integer">
      Number of days worked
    </ResponseField>
    
    <ResponseField name="diasFaltados" type="integer">
      Number of days absent
    </ResponseField>
    
    <ResponseField name="horasTotales" type="string">
      Total hours worked (HH:mm:ss)
    </ResponseField>
    
    <ResponseField name="promedioHorasDia" type="string">
      Average hours per day (HH:mm:ss)
    </ResponseField>
    
    <ResponseField name="retardos" type="integer">
      Number of times tardy

  </Expandable>
</ResponseField>

<ResponseField name="message" type="string">
  Success message: "Resumen de asistencia del mes"
</ResponseField>

### Example Request

## Export Compensation Report to Excel

<api>GET /opentime/compensaciones/excel</api>

Export the compensation report as an Excel file for payroll processing.

### Query Parameters

Same parameters as the JSON compensation report endpoint.

### Example Request

```bash cURL
curl -X GET "https://api.integra.example/opentime/compensaciones/excel?inicio=2026-03-01&fin=2026-03-31" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  --output compensaciones.xlsx
```

### Response

Returns an Excel file (`.xlsx`) with the following columns:
- Employee ID
- Employee Name
- Date
- Arrival Time
- Scheduled Start
- Departure Time
- Scheduled End
- Early Arrival Compensation
- Late Departure Compensation
- Total Compensation Time

## Get Attendance with Incidents

<api>GET /asistencia/incidencia</api>

Retrieve attendance records filtered by specific incident types (late arrivals, early departures, missed punches, etc.).

### Query Parameters

<ParamField query="inicio" type="string" required>
  Start date (yyyy-MM-dd format)
</ParamField>

<ParamField query="fin" type="string" required>
  End date (yyyy-MM-dd format)
</ParamField>

<ParamField query="empleadoId" type="integer" optional>
  Filter by employee ID
</ParamField>

<ParamField query="unidadId" type="integer" optional>
  Filter by unit ID
</ParamField>

<ParamField query="tipoIncidencia" type="string" optional>
  Filter by incident type (e.g., "RETARDO", "FALTA", "SALIDA_ANTICIPADA")
</ParamField>

### Example Request

```bash cURL
curl -X GET "https://api.integra.example/asistencia/incidencia?inicio=2026-03-01&fin=2026-03-07&tipoIncidencia=RETARDO" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### Response

```json
{
  "data": {
    "registros": [
      {
        "empleadoId": 1234,
        "nombreEmpleado": "María García",
        "fecha": "2026-03-03",
        "horaLlegada": "08:15:00",
        "horaSalida": "17:00:00",
        "incidencia": "RETARDO",
        "minutosRetardo": 15,
        "observaciones": "Tráfico"
      }
    ],
    "totalIncidencias": 1
  },
  "message": "Reporte de asistencia generado",
  "success": true
}
```

<Note>
This endpoint is useful for identifying patterns in attendance issues and generating disciplinary reports.
</Note>
